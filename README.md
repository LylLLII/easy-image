# 图片优化工具

一款简单易用的图片优化工具，支持调整尺寸和压缩图片。

## 功能特点

- 支持拖拽上传图片
- 可调整图片尺寸
- 支持多种压缩格式（JPEG、PNG、WebP）
- 可选择压缩质量
- 预设压缩方案（高质量、平衡、高压缩）
- 实时预览压缩效果

## 使用说明

1. 双击运行安装程序
2. 按照提示完成安装
3. 运行"图片优化工具"
4. 拖入图片或点击选择图片
5. 调整尺寸和压缩选项
6. 点击"调整尺寸并压缩"
7. 保存处理后的图片

## 系统要求

- Windows 7 及以上版本
- 64位操作系统
